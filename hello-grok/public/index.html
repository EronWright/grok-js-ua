Hello Grok Web Browser Example

This is a code demonstration on how you can use the JavaScript Client Library for Numenta's Grok engine. Part One details (with source code) how to initialize the Grok client object, create projects, streams, and models.

In each step of this example you can toggle the source code to see and understand how the JS Client Library is used, so be sure to click on the toggle source links to view the source for each step.
